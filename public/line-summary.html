สรุปรายวันส่ง LINE

สร้างข้อความสรุปสำหรับส่งกลุ่ม LINE ผู้ปกครอง
ตัวอย่างข้อความ
0 ตัวอักษร
วิธีใช้: กดปุ่ม "แชร์ไป LINE" → เลือกกลุ่มผู้ปกครอง → ส่ง
หรือกด "คัดลอก" → เปิด LINE เอง → ไปกลุ่ม → วาง (Paste) → ส่ง
💡 แนะนำใช้บนมือถือ เพราะกด "แชร์ไป LINE" จะเปิด LINE ให้เลือกแชทได้ทันที
